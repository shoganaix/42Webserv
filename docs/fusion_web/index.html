POST tests

POST should be rejected on /:

curl -i -X POST http://localhost:8080/

POST should be accepted on /post_body:

curl -i -X POST http://localhost:8080/post_body -d "12345"

POST body over 100 bytes should return 413:

curl -i -X POST http://localhost:8080/post_body -d "$(python3 -c 'print("a"*101)')"

CGI tests

POST to a .bla file should execute CGI:

curl -i -X POST http://localhost:8080/directory/youpi.bla -d "hello=world"

Optional GET test for CGI-related behavior:

curl -i http://localhost:8080/directory/youpi.bla

Directory tests

These routes are part of the official setup, but they may return empty bodies if the provided files are empty:

curl -i http://localhost:8080/directory/

curl -i http://localhost:8080/directory/nop/

Bad request tests

Invalid method:

printf 'BREW / HTTP/1.1\r\nHost: localhost:8080\r\n\r\n' | nc localhost 8080

Malformed request:

printf 'GARBAGE\r\n\r\n' | nc localhost 8080
